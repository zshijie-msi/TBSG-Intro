淘宝闪购新人自我介绍网页 v4

打开方式：
1. 保持整个 taobao_intro_site_v4 文件夹结构不变。
2. 双击 index.html，即可在浏览器中打开。

主要文件：
- index.html：网页结构和文案。
- styles.css：视觉样式和响应式布局。
- script.js：滚动出现、回到顶部、轻微 hover/鼠标动效。
- assets/：所有图片素材，包括头像、项目图、音乐会照片、二维码和 logo。
